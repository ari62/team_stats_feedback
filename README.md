# Feedback

This repository is used only for **in-app feedback** from [Team Stats](https://your-app-url-if-you-want).

- **Issues** are created automatically when users submit feedback (bugs, feature requests, or general feedback) from the app.
- Labels such as `bug`, `enhancement`, and `feedback` are applied based on the feedback type.
- No code lives here—issues only. Do not open PRs against this repo.
